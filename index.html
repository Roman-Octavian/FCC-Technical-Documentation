What is HTML?

HTML or HyperText Markup Language is a markup language that tells web browsers how to structure the web pages you visit. HTML consists of a series of elements, which you use to enclose, wrap, or mark up different parts of content to make it appear or act in a certain way.

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. HTML uses "markup" to annotate text, images, and other content for display in a Web browser. An HTML element is set off from other text in a document by "tags", which consist of the element name surrounded by "<" and ">".

What do you need?

To start working with HTML, there are a few requirements:

  • Basic computer literacy
  • Basic software installed on your machine
  • Basic knowledge of working with files

Notwithstanding that there are many ways to learn HTML, certain online resources such as freeCodeCamp or CodePen will allow you to learn and write HTML straight from your browser.

Basic structure of HTML

HTML elements are as seen in the image below mainly comprised of opening tags, optional attributes, the content itself, and closing tags.

The basic structure of an HTML file normally includes the <!DOCTYPE> at the very top, the <html> element right below encasing the rest of the code, and the <head> and <body> respectively. This basic structure would look something like this:

<!DOCTYPE html>
<html>
    <head>

    </head>
    <body>

    </body>
</html>

The <head> should contain a <title>, whereas the body has a structure of its own that usually contains:

  • a <header>
  • a <main> tag where the main content goes
  • a <footer> at the bottom of the page
HTML Metadata

The <head> is not displayed in the browser when the page loads as it mostly contains metadata. Metadata is data about the HTML itself. The most common elements included in the <head> are:

  • The <title> element, containing the overall title of the HTML, not to be confused with the <h1> heading.
  • Many <meta> tags which add a description, an author, images, theme colors, etc.
  • A favicon included with a <link> element.
  • And many others.
